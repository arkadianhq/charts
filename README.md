## Usage
* Add ArkadianHQ's helm repo

```shel
$ helm repo add arkadianhq https://charts.arkadianhq.com
```

* Check for updates - because that is a good thing to do :yum:

```shell
$ helm repo update arkadianhq
```

* Search for a chart

```shell
$ helm search repo arkadianhq
```
